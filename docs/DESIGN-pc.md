# v2note PC 端设计文档

> 核心理念：打开就是一张白纸。用户的文字是第一公民，所有功能为它服务。
> 结构从记录中涌现，AI 不说话除非被问。

---

## 一、全局架构

### 四场景 + 浮层 + 嵌入

```
v2note PC
├── 场景A：写作（默认，纯输入面板）        /write
├── 场景B：时间线（三栏浏览日记）          /timeline
├── 场景C：认知地图（网状图/思维导图）      /map
├── 场景D：目标（项目→目标→行动）          /goals
│
├── 常驻浮层：⚡️ 行动队列（右侧边栏 320px）
├── 常驻浮层：📋 每日回顾（居中弹窗 620px）
├── 常驻浮层：🔍 全局搜索（居中弹窗 520px）
└── 嵌入能力：💬 参谋对话（内嵌在触发位置）
```

### 菜单栏

```
┌────────────────────────────────────────────────────────────────────┐
│ [🦌] 念念有路  写作  时间线  地图  目标  ···  🔍 🎙 ⚡️行动 📋回顾 ⚙️ │
└────────────────────────────────────────────────────────────────────┘
```

- 固定顶部，高 44px
- 默认隐藏（opacity-0），鼠标进入顶部 48px 区域滑入，离开 400ms 淡出
- 左：路路 Logo(24px) + "念念有路"（font-serif font-bold text-bark） + 四场景按钮
- 右：工具按钮（搜索/语音/行动/回顾/设置）
- 当前场景按钮加粗高亮，其余半透明
- 📋 有新报告时显示鹿毛色小圆点
- 样式：bg-cream/95 backdrop-blur border-b border-brand-border

## 二、场景 A：写作面板

**任何时候打开 v2note，看到的就是这里。**

### 布局

```
┌──────────────────────────────────────┐
│          （菜单栏 hover 显现）         │
├──────────────────────────────────────┤
│                                      │
│  3月21日 周六                         │
│  ─────────────                       │
│                                      │
│  今天和张总开会，他说原材料             │
│  涨了15%，我觉得我们应该换             │
│  供应商......|                        │
│                                      │
│                                      │
├──────────────────────────────────────┤
│  Markdown │ 3行·47字  Ctrl+Enter 提交 │
└──────────────────────────────────────┘
```

- 居中 max-w-[680px]，充足上下留白
- 日期标题自动生成（font-serif text-sm text-[#9B8E82]）
- 等距字体（SF Mono / Fira Code），15px，行高 2 倍
- bg-cream，无边框，autofocus

### 输入方式

| 方式 | 触发 | 行为 |
|------|------|------|
| 打字 | 直接输入 | Markdown 语法实时渲染 |
| 语音 | 菜单栏 🎙 / Ctrl+Space | 录音→转写→文字插入编辑区 |
| 拖放 | 文件/图片拖入 | 虚线边框 + "松开即插入" + block 格式 |
| 粘贴图片 | Ctrl+V | [📷 image] block + 上传到 Ingest API |
| 粘贴 URL | Ctrl+V | [🌐 url] block + 提取预览 |
| 粘贴短文本 | <100字 | 直接插入（Think 权重） |
| 粘贴长文本 | ≥100字 | 弹确认"作为素材导入？" |

### 特殊输入

| 触发 | 行为 |
|------|------|
| `/` 行首 | 命令面板（13 命令 + Markdown 语法提示） |
| `@` | 结构列表（主题、目标），选择后插入关联标记 |
| `#` | 标签列表（已有 + 新建），用户标签权重 > AI 标签 |
| `#` + 空格 | Markdown H1 标题（不弹标签列表） |

### 命令面板

```
┌──────────────────────────────┐
│  🔍 搜索命令...               │
│  📋 /today     今日计划       │
│  🌙 /review    晚间总结       │
│  🗺 /map       认知地图       │
│  🎯 /goals     目标看板       │
│  ⚡ /actions   行动队列       │
│  💡 /think     决策分析       │
│  🔍 /search    搜索           │
│  📝 /timeline  时间线         │
│  ⚙️ /settings  设置          │
│  ── Markdown ──              │
│  H1 /h1       # 一级标题     │
│  H2 /h2       ## 二级标题    │
│  ☐  /todo     - [ ] 待办     │
│  >  /quote    > 引用         │
└──────────────────────────────┘
```

键盘上下选择 + Enter 确认 + 搜索过滤。

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Enter | 提交 → POST /ingest → "✓ 路路收到了"（1.5s） |
| Ctrl+S | 暂存到 localStorage（自动保存每 30s） |
| Tab | 呼出/收起涌现结构面板（右侧 280px） |

### 选中呼出工具栏

选中一段文字 → 浮出工具栏：

```
  ┌────────────────────────────────────┐
  │ 🔍 相关认知  💡 分析  📌 创建行动   │
  └────────────────────────────────────┘
```

### 底部状态栏

### 左栏：结构导航（200px，bg-sand）

- 涌现主题树 + 用户手动创建的结构
- 点击主题 → 中栏筛选
- "全部"按钮显示所有
- 选中项：bg-cream font-bold
- "+ 新主题"：手动创建目录，相关历史日记自动归入
- 右键：重命名 / 合并 / 删除
- 拖拽日记到主题 → 手动链接（高权重）
- 涌现新主题旁有微妙鹿毛色"新"小点
- 最多三层嵌套

### 中栏：日记流（flex-1）

筛选器：全部 | 语音 | 文字 | 图片 | 带文件 + 日期范围

每条日记卡片（推特式）：


- 原文（非 AI 处理后），line-clamp-4，点击展开
- 底部：主题标签 pill + 💬评论数 + 🔗关联数
- 日记流中**不插入** AI 观察卡片

## 四、场景 C：认知地图

### 工具栏

```
[🌐 网状图]  [🌲 思维导图]  自动布局  缩放适配  🔍搜索节点
```

bg-sand border-b border-brand-border h-10

### 网状图视图

- 白底 + 32px 网格背景
- 节点 = Cluster 卡片（白底 rounded-8px border-2）
  - 活跃：border-deer shadow-sm
  - 不活跃：border-brand-border opacity-70
  - 内容：名称（bold）+ 记录数（弱文字色）
- 连线 = 贝塞尔曲线，粗细代表跨主题日记频率
- 节点距离 = 语义关联度
- 双击节点 → 语义缩放（展开为内部日记卡片群）
- 拖线连接两个节点 → 创建手动 Bond

### 思维导图视图

```
       ┌─ 供应商管理 ─ 12条
  ┌─ 供应链 ────┤
  │             └─ 成本控制 ─── 8条
我的认知 ──┼─ v2note ────┬─ UI 设计 ──── 5条
  │              └─ 后端架构 ─── 9条
  └─ 个人 ──────┬─ 量化交易 ─── 6条
                └─ 成长 ─────── 4条
```

层级：项目 → 目标 → 主题 → 日记计数。经典树形展开，可拖拽重组。

### 右栏：节点详情（320px，点击节点展开）

📊 概览（记录数+活跃度+密度）→ 🎯 相关目标 → 📝 最近记录（last 5，可点击跳转时间线）→ 🔗 关联主题（relevance bar bg-sky/20）

## 六、浮层

### ⚡️ 行动队列（右侧边栏 320px）

```
┌─ ⚡️ 行动队列 ──────── ✕ ─┐
│  🔵 打给张总确认报价        │
│     供应链 · 10:00          │
│  ○ 审阅小李的报告           │
│     供应链 · 14:00          │
│     跳过2次                 │
│  ○ 完成 Digest prompt      │
│     v2note · 今天           │
│  ─────────────────────     │
│  已完成 (2) ▸               │
└────────────────────────────┘
```

- 第一项蓝色圆点高亮
- 点 ○ 标记完成
- 右滑 "稍后再说"
- 长按：原因选择（⏳🚧🔄）
- 跳过 5 次+：警告文字
- 折叠已完成区

### 📋 每日回顾（居中弹窗 620px）

Tab 切换：☀️晨间 / 🌙晚间

**晨间**：今日行动线 + "路路的发现"（偏差/模式/共振，高置信度陈述句，低置信度疑问句）

**晚间**：今日统计 + 洞察 + 最有价值记录 + 情绪追踪 + 反思引导（路路提一个问题 + 💬"想聊聊吗"）

**AI 报告层的唯一出口。所有系统洞察集中在这里，不散落到其他界面。**

### 🔍 全局搜索（Ctrl+K，居中弹窗 520px）

搜索输入 + 分类结果：记录 / 主题 / 目标 / 人物 / 命令。键盘上下导航。

### 💬 参谋对话（内嵌，不独占场景）

| 触发位置 | 上下文 |
|---------|--------|
| 时间线焦点栏 💬 | 当前日记 |
| 目标详情栏 💬 | 当前目标 |
| 每日回顾 💬 | 当前洞察 |
| 写作面板 /ask | 当前编辑内容 |
| 右键"问问路路" | 选中文字 |

路路头像作为 AI 头像。每条回复标注 [strike:xxx] 引用（可点击跳转）。支持 /munger 思维框架。对话保存为特殊类型日记。

## 九、渐进式解锁

| 日记数量 | 解锁功能 |
|---------|----------|
| 0+ | 写作面板、时间线 |

## 十、视觉风格

| 元素 | 规范 |
|------|------|
| 背景 | cream #FAF6F0（主）/ sand #F0E8DB（侧栏/面板） |
| 文字 | #2C2520（主）/ #6B5E52（次）/ #9B8E82（弱）/ #C4B8AC（隐） |
| 品牌 | bark #3D3228 / deer #C8845C / antler #A06B42 |
| 语义 | forest #5C7A5E / sky #7BA3C4 / dawn #E8A87C / maple #C45C5C |
| 字体 | Noto Serif SC（品牌）/ Noto Sans SC（正文）/ SF Mono（写作） |
| 圆角 | ≤ 12px |
| 阴影 | ≤ shadow-sm |
| 动画 | ≤ 200ms ease-out |
| 图标 | 优先 emoji |
| AI 口吻 | 路路（温暖、不催促、不评判） |

禁止：紫色渐变、冷灰色调、Inter/Roboto/Arial 字体
