# v2note 移动端设计文档

> 交互宪法：产品有两面。正面是一条线（执行），背面是一张网（思考）。用户 90% 时间看到线。

---

## 一、产品双面架构

```
          ┌──────────────┐
    上滑  │   🎙 录音入口  │  下拉
   ┌──────┤   （中间层）   ├──────┐
   │      └──────────────┘      │
   ▼                            ▼
┌──────────────┐    ┌──────────────┐
│  行动面板     │    │  认知地图     │
│  （执行面）   │    │  （思考面）   │
│  线性·聚焦   │    │  网状·发散   │
│  此刻该做什么 │    │  全局长什么样 │
│  单手10秒    │    │  双手慢探索   │
└──────────────┘    └──────────────┘
         │                  │
         └──── 长按穿越 ─────┘
```

三个手势，一个界面：
- **上滑**（底部）→ 行动面板
- **下拉**（顶部）→ 认知地图
- **长按穿越** → 两面切换

## 二、页面结构

### Level -1：纯净入口（默认态）

```
┌─────────────────────────────┐
│                             │
│                             │
│         16:28               │
│      3月21日 周六            │
│                             │
│    关联提示（5秒消失）        │
│                             │
│          🎙                  │
│                             │
│    ↑ 上滑行动面板            │
│    ↓ 下拉认知地图            │
└─────────────────────────────┘
```

- 暖奶色背景（bg-cream #FAF6F0）
- 大时间：font-serif text-5xl，暖色
- 日期：font-mono text-xs tracking-widest
- 录音 FAB 始终底部居中
- 右上角：🧠 Brain 图标（认知地图入口）+ 视图切换图标
- 无导航条、无侧边栏、无任何信息干扰

### Level 0：人生全景（认知地图）

```
┌─────────────────────────────┐
│  ← 我的认知世界              │
│                             │
│   ┌────────┐  ┌────────┐   │
│   │ 事业    │  │ 家庭    │   │
│   │ 28条    │  │ 12条    │   │
│   │ ●●●○   │  │ ●●○○   │   │
│   └────────┘  └────────┘   │
│                             │
│   ┌────────┐  ┌────────┐   │
│   │ 成长    │  │ 健康    │   │
│   │ 15条    │  │ 5条     │   │
│   │ ●●○○   │  │ ●○○○   │   │
│   └────────┘  └────────┘   │
│                             │
│         🎙 录音始终可用       │
└─────────────────────────────┘
```

- 2 列 grid，每个 Cluster 一张卡片
- 卡片：名称 + X条记录 + 活跃度圆点（4个，近期密度）
- **纯数据描述**，无判断性标签（"12条记录·近两周活跃"而非"🔴有矛盾"）
- 长按卡片 → 连接模式 → 拖线到另一个卡片创建手动 Bond
- 空态：路路完整形象 + "路路正在观察你的世界 · 继续记录，结构会自然涌现"

```
│ │ [帮我想想这个问题]      │  │
│ └─────────────────────────┘ │
│                             │
│ ┌─ 目标状态 ──────────────┐ │
│ │ 🎯 建立供应商评估体系    │ │
│ │ 方向 ██  资源 ▓░        │ │
│ │ 路径 ░░  驱动 ██        │ │
│ └─────────────────────────┘ │
│                             │
│ ── 认知时间线 ─────────────  │
│ 3/19 👁 原材料涨了15%       │
│      ⚖️ 我认为应该换供应商   │
│      🎯 让小李做成本对比     │
│ 3/15 👁 供应商交期又延迟了   │
│ 3/12 💡 供应链是系统性短板   │
└─────────────────────────────┘
```

四个区域（有数据时才显示）：
1. **认知模式**（bg-purple-50）：AI 发现的模式 + "这准确吗？[是][否]"确认
2. **对立观点**（bg-amber-50）：矛盾 Strike 对 + "帮我想想"入口
3. **目标状态**：四要素光谱条（方向/资源/路径/驱动），不用红绿灯，用填充程度
4. **认知时间线**：极性图标（👁蓝/⚖️橙/💡紫/🎯绿/❤️红）+ nucleus + 日期

## 三、行动面板（执行面）

### 触发

任何状态下拇指从底部上滑，半屏毛玻璃面板覆盖。

### 布局

```
┌─────────────────────────────┐
│ ┌─ 此刻（Now）────────────┐ │
│ │ 供应链评估               │ │
│ │ → 打给张总确认铝材报价    │ │
│ │   上次通话 3/15      📞  │ │
│ │   ← 跳过    完成 →      │ │
│ └─────────────────────────┘ │
│                             │
│ ── 今日线 ──────────────── │
│ ● 打给张总（供应链评估）     │
│ ○ 14:00 审阅成本对比        │
│ ○ 明天前 回复老王邮件        │
│ ◇ 记录本周供应链进展         │
│                             │
│  ● ○ ○ ○                   │
│  供应链 产品 团队 个人        │
└─────────────────────────────┘
```

### 此刻卡片交互

- **右滑 > 40%**：标记完成（绿色背景渐显）
- **左滑分叉**：滑到 30-40% 停住 0.3 秒
  - 继续左滑出去 = "稍后再说"（零摩擦）
  - 手指下拉 = "今天不做"，显示三标签：⏳等条件 / 🚧有阻力 / 🔄要重想
- **长按 500ms**：穿越到认知地图
- 每次滑动同时上报行为信号（fire-and-forget）

### 左滑行为学

左滑背后五种心理：现在不方便 / 还没准备好 / 不想面对 / 优先级变了 / 不该做。

后台追踪模式：
- 时间模式（总在早上跳过某类）
- 类别模式（只跳"跟人沟通"的）
- 关联模式（跳A后总做B）
- 频率模式（同一任务一周跳5次+）

跳过 5 次+，卡片安静出现："这件事已经在这里一周了。要聊聊吗？"

### 今日线

- ●：下一个要做的
- ○：有时间约束的后续
- ◇：灵活的

最多 4-5 项，超出折叠。



### 模糊意图场景

不知道该做什么时上滑，面板顶部显示"你有多少时间？[5分钟][15分钟][1小时]"，时间窗口重新筛选。

### 时间估算冷启动

- 前 30 天：不估时间，只按紧迫度排序
- 积累行为数据后：从右滑完成间隔学习同类任务时长
- 2 月+：个性化预判（"和张总通话平均 12 分钟"）

## 四、统一输入框

FAB 点击弹出底部 sheet：

```
┌─────────────────────────────────────┐
│  (drag handle)                      │
│  [附件预览区]                        │
│  [链接预览卡片]                      │
│  (多行文本输入区)                    │
│  开始记录...                         │
├─────────────────────────────────────┤
│  📎  │  输入或粘贴...        │  🎙  │
└─────────────────────────────────────┘
```

- **📎**（左）：拍照 / 相册 / 文件选择器
- **🎙**（右）：切换到独立录音沉浸界面，录完转写回填输入框
- 粘贴 URL 自动识别 → 预览卡片（标题+摘要）
- 发送 → POST /api/v1/ingest
- 所有附件默认 source_type='material'

## 六、冷启动

### 首次打开

播种对话（不显示空地图）：

```
┌─────────────────────────────┐
│       🦌 (路路 Logo)         │
│                             │
│    你好，我是路路             │
│    在开始之前，随便聊聊：      │
│                             │
│    [你最近在忙什么？]          │
│    [今年最想实现的一件事？]     │
│    [有什么想做但没开始的？]     │
│                             │
│    🎙 说说看  ⌨️ 打字        │
│    [跳过，直接开始]           │
└─────────────────────────────┘
```

### 渐进生长

1-3 条：种子散落
4-7 条：发芽，出现根系连接
8-10 条：幼苗，形成可辨认区域
10+：正常 Cluster 卡片墙

## 七、推送逻辑

| 层 | 时机 | 内容 | 语气 |
|----|------|------|------|
| 实时 | 录完音后 3 秒 | "关联到「供应链管理」" | 陈述 |
| 日常 | 晨间/到期时 | 今日行动 + deadline | 陈述 |
| 洞察 | 矛盾/模式发现时 | 认知洞察 | 疑问+确认 |

行为偏差用疑问语气：
> "最近很少提到运动——是计划有变，还是只是没录到？"
> [计划有变] [没录到] [在坚持，只是没说]

## 八、视觉风格

- 背景：暖奶色 #FAF6F0
- 文字：树皮色 #2C2520（主）/ #6B5E52（次）/ #9B8E82（弱）
- 强调：鹿毛 #C8845C / 鹿角 #A06B42
- 语义：森林绿 #5C7A5E / 天空蓝 #7BA3C4 / 晨光 #E8A87C / 枫红 #C45C5C
- 字体：Noto Sans SC（正文）/ Noto Serif SC（品牌标题）
- 路路：空状态、冷启动、侧边栏 Logo
- 圆角 ≤ 12px，阴影 ≤ shadow-sm，动画 ≤ 200ms ease-out
- AI 输出用路路口吻（温暖、不催促、不评判）
