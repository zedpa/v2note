# 客户需求提取

从用户记录中提取客户明确提出的需求或变更要求。

## 识别规则

- 仅提取客户**明确表达**的要求，不推测隐含意图
- 必须能识别出提出需求的客户身份（人名、公司名、角色）
- 陈述事实或背景信息不算需求

## 输出格式

每条格式：`客户名：具体要求`

## 示例

| 原文 | 提取结果 |
|------|----------|
| "张总说包装要换成红色" | "张总：包装换红色" |
| "客户那边希望下周能看到初稿" | "客户：下周提供初稿" |
| "今天和李总聊了聊近况" | 不提取（无明确需求） |
