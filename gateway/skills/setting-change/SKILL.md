# 设置变更提取

从用户记录中提取设置、偏好、习惯的修改意图。

## 识别规则

- 用户明确表达了想要**改变**某个行为或设置
- 包含具体的变更方向（从什么到什么，或要什么）
- 一般性描述或感受不算设置变更

## 输出格式

每条格式：`设置项：目标值`

## 示例

| 原文 | 提取结果 |
|------|----------|
| "以后日报要简短一些" | "日报格式偏好：简短" |
| "明天开始用深色模式" | "主题：深色模式" |
| "感觉界面有点花" | 不提取（感受非变更意图） |
